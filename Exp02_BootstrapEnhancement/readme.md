# WEEK – 2  
# EXPERIMENT – 2  
## 🛒 Enhancement of Shopping Cart Web Application using Bootstrap

---

## 🎯 Aim
To enhance the previously developed Shopping Cart web application using Bootstrap framework to achieve responsive design, improved layout, and professional UI components.

---

## 🎯 Objectives
- To understand Bootstrap framework  
- To apply Bootstrap grid system for responsive design  
- To use Bootstrap components like navbar, forms, cards, and tables  
- To reduce custom CSS usage  
- To improve mobile responsiveness  

---

## 💻 Software Requirements
- Operating System: Windows / Linux / macOS  
- Web Browser: Chrome / Firefox  
- Editor: Visual Studio Code / Notepad++  
- Bootstrap 5 (CDN)  

---

## 🖥 Hardware Requirements
- Minimum 4 GB RAM  
- Windows 10 or above  

---

## 📚 Theory
Bootstrap is an open-source front-end framework used to develop responsive and mobile-first websites. It provides pre-designed components such as navbar, grid system, cards, buttons, forms, and tables.

It helps in:
- Faster development  
- Consistent UI design  
- Responsive layouts  

---

## 📁 Project Structure
Exp02_BootstrapEnhancement/
│  
├── index.html  
├── register.html  
├── login.html  
├── catalog.html  
├── cart.html  

---

## ⚙️ Procedure
1. Create folder `Exp02_BootstrapEnhancement`  
2. Create required HTML files  
3. Add Bootstrap CDN link in all pages  
4. Replace plain UI with Bootstrap components  
5. Use navbar for navigation  
6. Use grid system for layout  
7. Use cards for products  
8. Use table for cart  
9. Run `index.html` in browser  
10. Verify responsiveness  

---

## 🧾 Description
This experiment enhances a shopping cart web application using Bootstrap to improve UI design and responsiveness. It includes Home, Login, Register, Product Catalog, and Cart pages.

---

## 📊 Output
- Responsive Home page with navbar  
- Styled Login and Register forms  
- Product catalog using Bootstrap cards  
- Cart page using Bootstrap table  
- Mobile-friendly layout  

---

## ⚠️ Result
The Shopping Cart web application was successfully enhanced using Bootstrap framework with improved UI and responsiveness.

---

## 👨‍💻 Conclusion
Bootstrap simplifies web development by providing ready-made components and responsive design utilities, making the application more professional and user-friendly.
