# EXPERIMENT – 1  
# 🛒 Exp01_ShoppingCart  
## Responsive Shopping Cart Web Application (HTML + CSS)

---

## 🎯 Aim
To design and develop a responsive shopping cart web application using HTML and CSS with Flexbox and Grid layout techniques.

---

## 🎯 Objectives
- To design a multi-page web application  
- To implement navigation bar using Flexbox  
- To use CSS Grid for product layout  
- To design responsive login and registration forms  
- To display cart items in table format  
- To make the web pages responsive for mobile and desktop  

---

## 💻 Software Requirements
- Operating System: Windows / Linux / macOS  
- Browser: Google Chrome / Mozilla Firefox  
- Editor: Visual Studio Code / Notepad++  

---

## 🖥 Hardware Requirements
- Minimum 4 GB RAM  
- Any modern processor system  

---

## 📚 Theory
HTML is used for creating structure of web pages and CSS is used for styling.

### Key Concepts Used:
- **Flexbox** → Used for navbar alignment and form centering  
- **CSS Grid** → Used for product catalog layout  
- **Media Queries** → Used for responsive design  
- **Tables** → Used for cart page  

This project simulates a basic e-commerce shopping system with multiple pages such as Home, Login, Register, Catalog, and Cart.

---

    Exp01_ShoppingCart

---

## ⚙️ Procedure
1. Create a folder named `Exp01_ShoppingCart`  
2. Create HTML files:
   - index.html  
   - login.html  
   - register.html  
   - catalog.html  
   - cart.html  
3. Create a folder named `css`  
4. Inside css folder create `style.css`  
5. Link CSS file in all HTML pages  
6. Design navigation bar using Flexbox  
7. Design hero section using Grid layout  
8. Design product catalog using Grid system  
9. Design login and registration forms  
10. Design cart page using HTML table  
11. Run `index.html` in browser
  
---
## 🧾 Description
This project is a responsive shopping cart web application built using HTML and CSS. It contains multiple pages with a consistent navigation system and responsive design.

---
## 📊 Output
- Home page with navigation bar  
- Login and Register forms  
- Product catalog using grid layout  
- Cart page using table format  
- Fully responsive design for mobile and desktop

## ⚠️ Result
The responsive shopping cart web application was successfully developed using HTML and CSS.
