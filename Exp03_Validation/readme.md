# 🧪 Experiment 03: Form Validation Using JavaScript

## 🎯 Aim
To design a web form and validate user input using JavaScript to ensure correct and complete data entry.

---

## 📁 Folder Structure
Exp03_Validation/
├── index.html
├── style.css
└── script.js

---

## 🧾 Description

This experiment demonstrates client-side form validation using JavaScript. The form checks user inputs before submission to ensure data correctness and avoid empty or invalid entries.

---

## ⚙️ Features

- ✔ Required field validation
- ✔ Email format validation
- ✔ Password length validation
- ✔ Confirm password matching
- ✔ Prevent form submission if invalid input

---

## 🧑‍💻 Technologies Used

- HTML5
- CSS3
- JavaScript

---

## ▶ How to Run

1. Open folder in VS Code  
2. Ensure all files are in same directory  
3. Open `index.html`  
4. Run using Live Server OR open in browser  

---

## 📌 Working Process

- User enters form details  
- JavaScript validates input fields  
- Shows error if invalid data is entered  
- Allows submission only if all inputs are correct  

---

## 📊 Output

- Valid form submission works successfully  
- Error messages shown for invalid input  
- Prevents incorrect data submission  

---

## ⚠ Note

- This is a frontend validation only  
- No database is used  
- Data is not stored permanently  

---

## 👨‍💻 Conclusion

The experiment successfully demonstrates form validation using JavaScript, ensuring proper user input handling on the client side.
